Highlights

This is a Gangrel Antitribu {{ gangrel_antitribu }} allies swarm deck starring {{ sebastien_goulet }}, who recruits a band of {{ shadow_court_satyr }}.

The core idea is to get {{ shadow_court_satyr }} with {{ earth_meld }}. This keeps them safe from combat: they can bleed every turn and combat ends if blocked. They can also be kept for defense and block quite safely.

Obfuscate {{ OBF }} provides excellent stealth, so the recruitements can be carried on without opposition.

A few {{ govern_the_unaligned }} can be used at superior Dominate {{ DOM }} to brin a sidekick in play for cheap, and {{ deflection }} is used in good numbers for bleed defense. Later, {{ govern_the_unaligned }} can be used to bleed, coupled with {{ conditioning }}, although {{ sebastien_goulet }} shouldn't be risked on too high a bleed in case of an {{ archon_investigation }}.

Tips & Tricks

Both {{ sebastien_goulet }} and {{ sebastien_goulet_adv }} are good stars here, and he even benefits from being merged, as he becomes a counter to other allies swarms, so both are played in even numbers.

{{ spying_mission }} can be used in case of bounce, to avoid bleeding the grand-prey and advantqging the prey.

{{ shadow_court_satyr }} is an expensive ally, so pool management is important: once {{ sebastien_goulet }} is out, the pool should be kepy to pay for the allies, and bringing a sidekick should rely on a {{ govern_the_unaligned }} to avoid spending too much too fast. Later in the game, once {{ sebastien_goulet }} is exhausted, {{ golconda }} can be used to sacrifice him and recoup the pool.

Variants

Even though {{ sebastien_goulet }} is a great star, it is possible to build a {{ shadow_court_satyr }} deck without him, see for example {{ desso_alastor_s_deck }} from 2015.
