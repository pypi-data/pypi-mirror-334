Highlights

This is a dreaded rush deck with {{ enkidu_the_noah }} as the star. He can rush and take down almost anyone, but also get some {{ raven_spy }} and block consistently with {{ sense_the_savage_way }}.

The combat module is dreadful, running so many {{ psyche }} that there's no escaping the beast. The occasional {{ claws_of_the_dead }} can help put some big foe in torpor, or even destroy them. A few dodges and damage prevention help protect from more active combat modules.

The archetype is called "multirush" because it is built to allow {{ enkidu }} to rush more than one vampire per turn. While he usually uses {{ forced_march }} or {{ freak_drive }} to unlock, this version interestingly focuses on {{ earth_meld }} instead. When he rushes, he can {{ earth_meld }} first, then continue the combat with {{ psyche }}, so he's unlocked and ready to rush anew if he gets one of the few action cards to do so

But {{ earth_meld }} also provide a fair bit of versatility since it can be used defensively too, avoid combats that are too dangerous, or providing much more unlock for intercepting large swarms.

Tips & Tricks

{{ andre_le_roux }} is a common inclusion in rush combat decks. Once the prey's vampires are removed, it might take a few turns to oust them. So {{ andre_le_roux }} can help, since he can bleed for 2 with his ability.

Variants

The most common variant is the Anarch variant, using {{ anarch_convert }} to make {{ enkidu }} Anarch, so he can spam {{ diversion }} and rush with {{ make_the_misere }}, but also benefit from interesting Master cards like {{ libertas }}, {{ powerbase_los_angeles }} and {{ infamous_insurgent }}. See {{ balazs_kaloczkai_s_deck }}.

A Potence {{ POT }} version of the archetype is also possible, using {{ immortal_grapple }} over {{ psyche }}, like in {{ john_b_whelan_s_deck }}.
